FinanceIQ – AI Powered Personal Finance Manager

FinanceIQ is a modern web-based personal finance management application designed to help users track expenses, manage budgets, analyze spending habits, and improve financial health using AI-powered insights.

##  Features

* 🔐 User Login & Profile Management
* 💸 Income and Expense Tracking
* 📊 Interactive Analytics Dashboard
* 🎯 Budget Planning & Monitoring
* 🤖 AI Financial Coach
* 🧠 AI Spending Analyzer
* ❤️ Financial Health Score
* 🔔 Smart Budget Alerts
* 📅 Expense Heatmap Visualization
* 📈 Savings Progress Tracking
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* Local Storage
